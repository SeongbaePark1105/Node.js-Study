## 3. URL의 이해

```
http://opentutorials.org:3000/main?id=HTML&page=12
```

```
http
```

- Protocol
  - 통신 규칙이라고 함.
  - 즉, 사용자가 서버에 접속할 때 어떤 방식으로 통신하려는 지

```
opentutorials.org
```

- Host(Domain)
  - 인터넷에 접속되어있는 각각의 컴퓨터를 Host

```
3000
```

- Port

  - 포트 번호

  - 한 대의 컴퓨터안에 여러 서버가 있을 수 있는 데 클라이언트가 어떤 서버와 통신할지 

    정해주는 번호

  - 기본 값은 80

```
main
```

- Path
  - 컴퓨터안에 있는 어떤 폴더안에 어떤 파일인지 알려줌

```
id=HTML&page=12
```

- query string
  - ?로 시작 (약속)
  - 값과 값은 &로 약속
  - 구분은 =로 약속
